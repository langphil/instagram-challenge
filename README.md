Instagram Challenge
===================
Instagram knock off

## Functionality
- [x] User signup / in / out / edit / delete
- [x] User can post photos
- [x] Home feed of photos
- [x] User account feed

## Installation
* git clone git@github.com:langphil/instagram-challenge.git
* bundle install
* bin/rails db:create
* bin/rails db:migrate
* bin/rails server
* http://localhost/3000

* rspec (runs tests)
* rubocop (code quality)
